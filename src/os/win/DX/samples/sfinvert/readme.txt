Sonic Foundry Invert Plug-In (SFINVERT.DLL)

The Sonic Foundry Invert Plug-In demonstrates the most basic in-place
transform possible. There are no property pages, etc.

Files contained in this example:

    readme.txt      - this file
    sfinvert.h      - classes and defines for the transform
    sfinvert.cpp    - transform (the real work) is in here
    sfinvert.def    - generic linker definition file
    sfplugin.cpp    - DirectX 'glue' (usually doesn't need changes)
    resource.h      - resource id's
    sfinvert.rc     - version information
    makefile        - DirectX makefile
    _depend         - makefile dependencies
